# web-element-size

A demo app showcasing the `ElementSize` type from `@inglorious/web`.

An `ElementSize` entity observes a board element via a CSS selector and reports
its live width and height. Add or remove items and the board resizes to fit its
content, while the readout next to it updates in real time — a size change that a
window `resize` listener would never catch.

Quick start:

```sh
pnpm install
pnpm dev
```
