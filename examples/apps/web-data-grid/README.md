# web-data-grid

A tiny demo that shows an example usage of the data grid component from Inglorious UI.

Quick start:

```bash
# from repository root
cd examples/apps/web-data-grid
pnpm install
pnpm dev
```

Open the app in your browser (Vite normally serves at http://localhost:5173) and inspect `src/product-table/product-table.js` and `src/store/entities.js` for the implementation and how the `data-grid` type is wired.
