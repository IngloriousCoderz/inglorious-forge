# web-combobox

A demo app showcasing the `combobox` component from `@inglorious/web`.

This example demonstrates three common use cases for a combobox/dropdown component:

- A simple, searchable single-combobox box.
- A multi-combobox box allowing for multiple comboboxions.
- A single-combobox box that fetches its options from a remote source asynchronously.

Quick start:

```bash
# from repository root
cd examples/apps/web-combobox
pnpm install
pnpm dev
```
