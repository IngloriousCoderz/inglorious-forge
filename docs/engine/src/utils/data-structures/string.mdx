# String

Functions that deal with a single string.

## `toCamelCase`

Converts `kebab-case` or `PascalCase` identifiers into `camelCase`.

```javascript
import { toCamelCase } from "@inglorious/utils/data-structures/string.js"

toCamelCase("my-element") // myElement
toCamelCase("MyElement") // myElement
```
