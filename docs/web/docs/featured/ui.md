# UI Primitives ([Inglorious UI](https://inglorious.dev/ui))

Ready-made UI primitives live in **[Inglorious UI](https://inglorious.dev/ui)**. It provides a full design system (controls, data display, navigation, feedback, layout, surfaces) built with the same entity/type model.

Use [Inglorious UI](https://inglorious.dev/ui) when you need polished, reusable UI elements without building them yourself. You can still override `render()` and `renderItem()` just like any other type.
